# Salmo do Dia

Projeto simples e minimalista para leitura diária de salmos,
integrado à rotina de oração da manhã e da noite.

## Propósito

Criar um ritmo estável de oração com um ciclo fixo de salmos,
permitindo familiaridade e aprofundamento ao longo do ano.

Não é um aplicativo.
Não é uma plataforma.
É apenas um espaço silencioso para o salmo do dia.

## Funcionamento

- O sistema alterna automaticamente entre:
  - **Salmo da Manhã** (antes das 18h)
  - **Salmo da Noite** (após as 18h)

- O ciclo é calculado com base na data inicial definida no código.
- Os salmos giram em repetição contínua.
- A repetição intencional permite maior interiorização.
